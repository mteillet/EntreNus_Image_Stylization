In order to run the python standalone version, make sure you have python 3.7+, opencv-python, matplotlib, numpy, and pyQt5

In order to use the program, launch the ENTRE_NUS_STYLIZATION_ALPHA_00_01.py file it will open a GUI allowing you to convert your render passes

Right now the python version is only intended to be used when you can't access Nuke. Therefore, it is much longer and offers a lot less possibilities than what you can do with the blinkscrips provided
